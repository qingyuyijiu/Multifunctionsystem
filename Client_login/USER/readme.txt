因此无需将.so文件放入/lib文件夹

文件夹描述:
1.COOE:源代码
2.LIB:库
3.USE:在此处上编译。指令./make.sh
4.BIN:可执行文件在此处生成
5.source:程序运行要使用的资源，放到开发板根目录即可


名称:青鱼OS客户端
功能:相册，五子棋，画板,视频播放器，环境检测功能，手电筒，登陆界面实现登陆注册，语音识别
执行方法:把LIB中的库文件放入开发板的\lib中，再把sourcr放到根目录下,执行.\main就可以运行了
1.相册：点击界面上的相册，进入相册的多图模式(4张图片)，向上或者向下滑动切换图片，图片的相对位置不会因为滑动而改变，
点击图片即可进入单图模式，再次点击图片进行图片特效展示，向上或向下滑动切换图片，向左滑动退出单图模式，再次左滑动退出相册
2.五子棋：点击界面上的五子棋，进入五子棋模式，点击屏幕上的格子，可以进行落子，游戏结束后可以重新开始，也可以退出游戏，黑棋先手
3.画板：点击界面上的画板，进入画板模式，可以调整画板大小，点击上方不同颜色的调色板可以切换到对应的颜色，同时右下角会实时显示画笔的大小和颜色。
画满画板后，可以点击右上角的按钮进行清屏操作。
4.视频播放器：点击界面上的视频播放器，进入视频播放器，首先会在指定的目录加载视频文件，加载完毕后开始播放第一个视频，开始播放后会初始化音量到合适大小，
功能：暂停/播放，快进/后退，音量调节，向右滑动倍数播放，向左滑动恢复，下方进度条会显示当前播放进度。点击进度条可以跳转到指定位置播放。
5.环境检测，检测周围的亮度，温度，湿度，和烟雾浓度，当亮度达到一定亮度大于20灯D9就会灭，当烟雾浓度超过150时蜂鸣器就会报警
6.手电筒，点击控制灯D8的暗灭
7.登陆界面，用户使用账号和密码进行登陆，通过网络把登陆账号密码发送给服务器，服务器对用户账号和密码进行查询看是否在数据库上，有则可登陆，无就不能登陆，
用户还可以输入手机号进行注册，要有11为，然后发送给客户端，服务器随机生成一个账号发送给客户端，同时把该用户的手机号，账号，密码记录在服务器中。
8，日志，记录客户端登陆和注册的信息。
9，语音识别，登陆成功后，客户端可通过喊小鱼小鱼，唤醒语音助手，通过语音识别可以打开桌面上的所有软件，还可以控制灯泡开光
